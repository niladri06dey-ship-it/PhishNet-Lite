🚀 Quick Start (3 steps)
1. Install Python dependencies
bashpip install flask flask-cors
2. Start the backend
bashpython app.py
→ Server starts at http://127.0.0.1:5000
3. Open the frontend
Just double-click index.html in your file manager — or serve it with:
bashpython -m http.server 8080
# open http://localhost:8080
